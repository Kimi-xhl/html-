# HTTP 请求方法
1. GET 请求指定的页面信息，并返回实体主体
2. POST 向指定资源提交数据进行处理请求（例如提交表单或者上传文件）。数据被包含在请求体中。POST 请求可能会导致新的资源的建立和/或已有资源的修改。
